# Neo4j Enterprise Studio

Neo4j Enterprise Studio is a self-hosted platform that brings collaborative graph tooling to your organization. It allows users to query, explore, and visualize Neo4j data in a shared context through a single interface.

Enterprise Studio provides:

-   A visualization layer for multiple database deployments and an interface for users to engage with graph data.

-   A centralized location for storing and sharing tool-specific assets such as queries, dashboards, and Perspectives.


Enterprise Studio brings a collaborative environment to self-managed Neo4j deployments, similar to the fully managed cloud solution, [Neo4j Aura](https://neo4j.com/docs/aura).
