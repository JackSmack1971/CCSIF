# APOC Core 2026.06 Documentation

<table><tbody><tr><td class="icon"><i class="fa icon-note" title="Note"></i></td><td class="content"><div class="paragraph"><p>This manual covers the documentation for APOC Core. For APOC Extended, go to the <a href="https://neo4j.com/labs/apoc/5/overview">APOC Extended manual</a>.</p></div></td></tr></tbody></table>

The guide covers the following areas:

-   [Introduction](https://neo4j.com/docs/apoc/current/introduction/) — An Introduction to the APOC library.

-   [Installation](https://neo4j.com/docs/apoc/current/installation/) — Installation instructions for the library.

-   [APOC and Cypher versions](https://neo4j.com/docs/apoc/current/cypher-versions/) — APOC and different Cypher versions.

-   [Built-in Help](https://neo4j.com/docs/apoc/current/help/) — Built-in help in the library.

-   [Configuration Options](https://neo4j.com/docs/apoc/current/config/) — Configuration options used by the library.

-   [Security Guidelines](https://neo4j.com/docs/apoc/current/security-guidelines/) — Guidelines on securing the APOC library, and its environment.

-   [Procedures & Functions](https://neo4j.com/docs/apoc/current/overview/) — A list of all APOC procedures and functions.

-   [Import](https://neo4j.com/docs/apoc/current/import/) — A detailed guide to procedures that can be used to import data from different formats including JSON, CSV, and XLS.

-   [Export](https://neo4j.com/docs/apoc/current/export/) — A detailed guide to procedures that can be used to export data to different formats including JSON, CSV, GraphML, and Gephi.

-   [Graph Refactoring](https://neo4j.com/docs/apoc/current/graph-refactoring/) — A detailed guide to procedures that can be used to refactor graphs.

-   [Graph updates](https://neo4j.com/docs/apoc/current/graph-updates/) — A detailed guide to procedures that can be used to apply graph updates.

-   [Data Structures](https://neo4j.com/docs/apoc/current/data-structures/) — A detailed guide to procedures and functions, that can be used to work with data structures.

-   [Temporal (Date Time)](https://neo4j.com/docs/apoc/current/temporal/) — A detailed guide to procedures that can be used to format temporal types.

-   [Mathematical Operations](https://neo4j.com/docs/apoc/current/mathematical/) — A detailed guide to procedures and functions that can be used for mathematical operations.

-   [Advanced Graph Querying](https://neo4j.com/docs/apoc/current/graph-querying/) — A detailed guide to procedures that can be used for advanced graph querying.

-   [Comparing Graphs](https://neo4j.com/docs/apoc/current/comparing-graphs/) — A detailed guide to procedures that can be used to compare graphs.

-   [Conditional Cypher Execution](https://neo4j.com/docs/apoc/current/conditionals/) — A detailed guide to procedures that can be used for to execute Cypher conditionally.

-   [Cypher Execution](https://neo4j.com/docs/apoc/current/cypher-execution/) — A detailed guide to procedures that can be used for Cypher scripting.

-   [Virtual Nodes & Relationships (Graph Projections)](https://neo4j.com/docs/apoc/current/virtual/) — A detailed guide to procedures that can be used to create virtual nodes and relationships.

-   [Background Operations](https://neo4j.com/docs/apoc/current/background-operations/) — A detailed guide to procedures that can be used for background job management.

-   [Cypher initializer](https://neo4j.com/docs/apoc/current/operational/) — A detailed guide to operational procedures.

-   [Schema Information](https://neo4j.com/docs/apoc/current/schema/) — A detailed guide to indexing procedures.
