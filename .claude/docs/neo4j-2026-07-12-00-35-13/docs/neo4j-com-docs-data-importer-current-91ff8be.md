# Neo4j Data Importer

Neo4j Data Importer is a UI-based tool for importing data into Neo4j. This functionality exists both as a standalone tool as well as a service in the fully managed Neo4j Aura console. This manual covers the standalone tool. For information about the import tool in the Aura console, see [Aura Documentation → Import](https://neo4j.com/docs/aura/import/introduction).

This manual covers the following areas:

-   [Introduction](https://neo4j.com/docs/data-importer/current/introduction/) — An introduction to the Neo4j Data Importer.

-   [Overview](https://neo4j.com/docs/data-importer/current/overview/) —  A visual tour of the Data Importer UI.

-   [File provision](https://neo4j.com/docs/data-importer/current/file-provision/) — How to provide files for import.

-   [Modeling](https://neo4j.com/docs/data-importer/current/modeling/) — Guidelines to model your data.

-   [Mapping](https://neo4j.com/docs/data-importer/current/mapping/) — How to map your data to the model.

-   [Indexes and constraints](https://neo4j.com/docs/data-importer/current/indexes-constraints/) — Optimize query performance.

-   [Import](https://neo4j.com/docs/data-importer/current/import/) — Preview and import your data.

-   [Other options for data import](https://neo4j.com/docs/data-importer/current/import-others/) — Other ways to import data.
