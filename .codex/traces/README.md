# traces

Generated by `.codex/hooks/post-tool-use.sh` and `.codex/hooks/stop.sh` via
`.codex/hooks/lib/trace-writer.js`. One `<YYYY-MM-DD>.jsonl` file per UTC day,
appended to on every `PostToolUse` and `Stop` hook firing.

Schema and consumption contract: see
`.claude/skills/self-improve/references/input-discovery.md` (section 4).

Known limitation: `error_class` values other than `tool_failure` and `null`
are not mechanically derived by these hooks - they require semantic judgment
the hooks don't have visibility into.

This directory is generated telemetry, not source. Do not hand-edit these files.
