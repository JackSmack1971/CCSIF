In the HINDSIGHT Memory Architecture, the Three Core Operations—Retain, Recall, and Reflect—are the active mechanisms that animate the system's memory bank 1, 2. While the four logical networks (World, Experience, Opinion, and Observation) define where information structurally lives, these three operations govern exactly "how information is added, accessed, and updated" over the lifespan of the AI agent 1, 2.
Together, they form a continuous, unified cognitive loop 3, split across the architecture's two main processing components, TEMPR and CARA 4:
1. Retain (Managed by TEMPR)The Retain operation is responsible for ingesting new conversational transcripts and seamlessly integrating them into the structured memory bank 5.
* It extracts raw input into comprehensive "narrative facts" and classifies each one into its appropriate network 5, 6.
* It resolves canonical entities and maps the new facts into the underlying memory graph by constructing explicit temporal, semantic, entity, and causal links 5, 6.
* As new evidence is retained, this operation also triggers background processes, such as merging background biographical details and using the reinforcement mechanism to update the confidence scores of the agent's existing subjective beliefs 6, 7.
2. Recall (Managed by TEMPR)When a user asks a question, the Recall operation acts as an agent-optimized retrieval interface to pull the most relevant information from the memory bank 7, 8.
* Rather than relying on just one search method, it executes a four-way parallel retrieval pipeline combining semantic vector search, BM25 keyword search, spreading-activation graph traversal, and temporal filtering 7, 9.
* Crucially, this operation takes a specific "token budget" as a parameter. It ranks all the retrieved memories (using Reciprocal Rank Fusion and neural reranking) and returns only the top most relevant facts that fit within the downstream LLM's context window limit 7, 10.
3. Reflect (Managed by CARA)The Reflect operation is the final step where the agent formulates its response and evolves its worldview 4, 11.
* It takes the objective facts and synthesized summaries retrieved by the Recall operation and filters them through the agent's specific behavioral profile (its configured levels of skepticism, literalism, empathy, and bias strength) 11, 12.
* It generates a final response whose tone and reasoning are explicitly shaped by these preferences 11.
* Simultaneously, the Reflect operation uses the newly retrieved context to form new subjective opinions (complete with confidence scores), storing them back into the Opinion Network for future use 11, 13.
Ultimately, these three operations define a continuous lifecycle for long-lived AI agents: new experiences are strategically retained into a structured graph, precisely recalled when a query demands it, and dynamically reflected upon to generate consistent answers and update the agent’s evolving identity 3.
