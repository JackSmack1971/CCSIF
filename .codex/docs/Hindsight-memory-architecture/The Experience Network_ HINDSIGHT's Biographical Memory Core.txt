In the HINDSIGHT architecture, the Experience Network is one of the four logical networks (alongside the World, Observation, and Opinion networks) designed to enforce the system's core principle of epistemic clarity 1. While the other networks handle external environmental facts, synthesized summaries, or subjective beliefs, the Experience Network specifically serves as the repository for the agent's own biographical history 2.
Within the larger context of the four-network memory bank, the sources highlight the following characteristics of the Experience Network:
* First-Person Biographical Actions: The Experience Network stores information about the AI agent's own past actions, experiences, and recommendations, and these memories are explicitly written in the first person 2. For example, a typical fact stored in this network would be: "I recommended Yosemite National Park to Alice for hiking" 3.
* A Pillar of Objective Evidence: Along with the World Network (which stores facts about the external environment), the Experience Network makes up the strictly objective portion of the agent's memory 1. While it deals with the self rather than the world, it strictly records what the agent actually did or said, keeping it completely separate from the subjective judgments stored in the Opinion Network 2, 4.
* Foundation for Synthesis and Reflection: The objective biographical facts stored in the Experience Network act as a foundational data source for HINDSIGHT's higher-level cognitive functions. For instance, the system's Observation Network draws directly on underlying facts from both the Experience and World networks to synthesize concise, preference-neutral summaries about specific entities 2. Additionally, during the Reflect operation, the CARA reasoning component retrieves these past experiences to use as grounded evidence when forming new subjective opinions or answering user queries 5.
By dedicating a specific structural space to the agent's own history, HINDSIGHT ensures that a long-lived AI can maintain a clear, auditable record of its past interactions and decisions without confusing its own actions with general world knowledge 2, 6.
